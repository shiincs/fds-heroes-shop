# 넥센 히어로즈 샵

넥센 히어로즈 굿즈샵 사이트를 클론해서 만든 사이트입니다.

## 기능

1. 상품 목록 출력 (카테고리별 필터링)
2. 상품 상세정보 (상품 옵션 및 수량 선택, 장바구니 담기, 구매내역 담기)
3. 장바구니 확인 (상품 수량 수정)
4. 구매내역 확인
5. 회원가입 / 로그인 / 로그아웃

기능은 JSON Server API 와의 비동기 통신으로 데이터를 주고받는 형식으로 개발했습니다.
